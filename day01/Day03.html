การเขียนCSS


เขียนแบบ ความหมาย
Inline
 คือการเขียนแทรกลงไปในบรรทัดนั้นๆ ของ tag [X]HTML เลย เช่น style=”font-size: 0.9em; width: 200px; border: solid 1px #000000”>
Internal
 การเขียนแบบ Internal (embedded) คือเขียนแยก CSS ออกจาก tag [X]HTML แต่อยู่ในหน้าเดียวกันกับไฟล์ [X]HTML โดยแทรก css
External
 การเขียน แยก css ออกมาเป็นไฟล์ .css ต่างหาก แล้วนำเข้าไปใช้ใน หน้าเว็บด้วย การเรียกโดย ใช้ โดยการพิมพ์ <#link rel="stylesheet" href="externalcss.css" type="text/css" />
